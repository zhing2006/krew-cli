# Phase 12: 交互打磨

> 目标：提升 TUI 交互体验，补全所有产品细节。

## 实现内容

- **@ 补全**：输入 `@` 后弹出 Agent 名称列表
- **/ 补全**：输入 `/` 后弹出命令列表
- **思考过程显示**：`💭` 折叠显示 ThinkingDelta 内容
- **Ctrl+C 中断**：中断当前 Agent 输出，保留已接收内容
- **Web Search 显示**：搜索结果 `[n]` 脚注格式
- **流式中断提示**：Agent 回复不完整时提示用户

## 验收标准

- `@` 和 `/` 触发补全菜单
- Ctrl+C 可中断流式输出
