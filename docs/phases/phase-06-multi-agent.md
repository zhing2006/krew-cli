# Phase 6: 多 Agent 协作

> 目标：实现 @all 广播、串行执行、上下文共享，多个 Agent 在同一会话中协作。

## 实现内容

- **@all 串行执行**：按 `reply_order` 顺序执行每个 Agent 的完整回合
- **上下文共享**：前一个 Agent 的回复追加到消息历史，后续 Agent 可见
- **@name 单独对话**：只有指定 Agent 回复，但消息对所有 Agent 可见
- **无前缀延续**：发给上一个回答者，若无上一个回答者则提示用户指定
- **Token 累计**：`session.total_tokens_used` 累加所有 Agent 用量
- **@all 错误隔离**：某个 Agent 失败不影响其他 Agent 继续

## 验收标准

```txt
you> @all 自我介绍一下

[gpt] GPT-5.2:
  你好！我是 GPT-5.2...

[opus] Claude Opus:
  你好！我是 Claude Opus，GPT 刚才也做了自我介绍...（能看到 GPT 的回复）

you> 继续说说你的特长（→ 发给 opus，因为它是上一个回答者）
```
